杭州电子科技大学（杭电）的“智能科学与技术”专业（简称“智科”）是计算机学院或人工智能学院下设的热门专业之一。该专业注重人工智能、机器学习、数据处理、智能系统等方向的培养。

以下是**大二学年**（即第三、第四学期）通常开设的主要课程，具体可能因年份和培养方案微调略有不同，但整体结构较为稳定：

---

### **大二上学期（第3学期）主要课程：**

1. **数据结构**
   - 核心课程，学习线性表、栈、队列、树、图等结构，为算法和编程打基础。

2. **面向对象程序设计（C++ 或 Java）**
   - 深化编程能力，掌握类、继承、多态等 OOP 概念。

3. **离散数学**
   - 包括数理逻辑、集合论、图论、代数结构等内容，是计算机理论的基础。

4. **数字逻辑电路**
   - 学习门电路、组合/时序逻辑、触发器、计数器等，为后续硬件课程铺垫。

5. **概率论与数理统计**
   - 重点学习概率分布、期望、方差、假设检验等，为机器学习提供数学支持。

6. **大学物理（下）**（部分方向仍需修读）
   - 含电磁学、光学等内容，视培养方案而定。

7. **工程图学 / 计算机导论拓展**（视具体安排）

---

### **大二下学期（第4学期）主要课程：**

1. **算法设计与分析**
   - 学习贪心、动态规划、分治、回溯等经典算法，提升编程与解题能力。

2. **操作系统**
   - 进程管理、内存管理、文件系统、I/O 系统等核心内容。

3. **计算机组成原理**
   - 学习冯·诺依曼体系、CPU 结构、指令系统、存储系统等。

4. **Python 程序设计 / 高级编程实践**
   - 强化 Python 在数据分析、AI 中的应用能力。

5. **线性代数**
   - 矩阵运算、向量空间、特征值等，AI 和机器学习的重要数学基础。

6. **复变函数与积分变换**（部分方向开设）
   - 视专业方向要求，用于信号处理等后续课程。

7. **智能科学导论 / 人工智能导论**（部分班级开设）
   - 初步了解 AI 发展、基本概念与应用。
